## Coverage Investigation

### Coverage Gaps
The ROADMAP explicitly lists planned harnesses for `variant-annotation`, `clinpgx`, `gwas-prs`, `clinical-trial-finder`, `target-validation-scorer`, `genome-compare`, `methylation-clock`, and `wes-clinical-report` (P1 tier), none of which have current harness coverage. The existing `clinical-variant-reporter` harness only covers Phase 1 structural checks, not the planned Phase 2 correctness tests.

### Test Case Updates Needed
The `pharmgx-reporter` shows a systematic `disclosure_failure` pattern (9 of 14 failures), indicating test cases must enforce explicit disclosure of scope limitations for CYP2D6, CYP3A5, and UGT1A1. The `equity-scorer` requires updates to mandate estimator labels (`fst_mislabeled`) and handle haploid edge cases (`edge_crash`).

### New Patterns
The `stub_silent` category in `bio-orchestrator` (3 failures) reveals a new safety/honesty pattern: stub skills should warn when invoked. The `susie_nonconvergence_suppressed` in `clawbio-finemapping` indicates a need to add convergence-check test cases that suppress PIP outputs when SuSiE fails to converge.

### ROADMAP Adjustments
The high `disclosure_failure` rate in `pharmgx-reporter` (61.1% pass) elevates the priority of the planned `clinpgx` harness (P1-3) and the `pharmgx` expansion (P1-E7) to address honesty gaps in pharmacogenomics. The `clinical-variant-reporter` Phase 2 (P1-1) should be prioritized given its current `data_source_version_missing` failure.

### Action List
1. Update `pharmgx-reporter` test cases to enforce disclosure of scope limitations for CYP2D6, CYP3A5, and UGT1A1.
2. Add `stub_silent` as a critical failure category in `bio-orchestrator` and fix routing to warn on stub skills.
3. Implement the `clinpgx` harness (P1-3) to cover pharmacogenomics annotation gaps.
4. Update `equity-scorer` to require estimator labels and handle haploid edge cases.
5. Add convergence-check test cases to `clawbio-finemapping` to suppress PIPs when SuSiE does not converge.
