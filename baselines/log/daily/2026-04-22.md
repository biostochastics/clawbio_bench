Each suggests a need for additional test cases to probe the boundaries and root causes of these failures more thoroughly.

**New category patterns**: The presence of `transcript_selection_error` and `gene_disease_context_missing` suggests a pattern of annotation and context failures in clinical reporting. This points to a need for new test cases that validate the completeness and accuracy of genomic context (e.g., gene-disease validity, transcript selection logic) across all clinical skills.

**ROADMAP adjustments**: The current coverage confirms the high priority of the P1 clinical safety harnesses, as existing clinical sub-harnesses (`cvr-variant-identity`, `gwas-prs`) are already generating findings. The planned P1-E8 re-verification of ClawBio's pass rate post-remediation is critical, as current findings indicate persistent issues. The P2 harnesses for population integrity (e.g., `claw-ancestry-pca`) remain a clear gap.
