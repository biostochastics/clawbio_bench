# DERIVATION: With z=[0,0,0], R=I, n=100, L=3, the residual r_l = 0 for every effect, so log_abf is uniform across variants and each alpha row stays uniform at 1/p=1/3. PIP_i = 1 - prod_l(1 - alpha_{l,i}) = 1 - (2/3)^3 = 19/27 ≈ 0.7037. A default-configured susieR::susie_rss(z=0, R=I, n=100, L=3) exhibits the same algebraic pathology because the Single Effect Regression (SER) submodel has no "null" component and must place unit mass per effect row. susieR mitigates the downstream damage with the optional `null_weight` parameter (adds an explicit null alternative to each SER prior) and via `susie_get_cs`'s purity + `min_abs_corr` pruning, which drops the bogus credible sets before they reach the user. ClawBio implements neither mechanism, so the raw 0.7-per-variant PIP flows straight through to callers. The audit criterion for this test is intentionally conservative: any max PIP > 0.1 on a z≡0 locus is an unambiguous auditable red flag for downstream consumers that inspect PIP directly (not via credible sets).
