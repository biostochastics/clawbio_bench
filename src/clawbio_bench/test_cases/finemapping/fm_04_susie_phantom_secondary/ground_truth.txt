# DERIVATION: Single causal at variant 0 (z=5), null at variant 1 (z=0), R=I, n=100, L=2. After the first IBSS sweep alpha_0 ≈ [1.0, ~0]; the residual seen by effect row 2 is z - R @ (alpha_0*mu_0) ≈ [5 - 4*0.99998, 0] ≈ [1e-4, 0], which is essentially null for variant 0 but the SER still has to put unit mass SOMEWHERE on that row. ClawBio's implementation lets alpha[1] drift into a ~0.7/~0.3 split, producing PIP[1] ≈ 1 - (1-0)*(1-0.298) ≈ 0.298. susieR::susie_rss with `null_weight >= 0.1` suppresses this by adding an explicit null alternative to each SER prior, so the second effect row's posterior mass lands on "no effect" instead of being forced onto a real variant. susieR without null_weight (default) produces similar raw PIPs but then prunes the impure credible set via susie_get_cs(min_abs_corr=0.5). ClawBio does neither. The auditable criterion: on a one-signal locus with L>1, any secondary PIP > 0.05 is a published-value red flag, regardless of credible-set handling.
